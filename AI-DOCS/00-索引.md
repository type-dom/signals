## 🔑 核心概念速查

### Signal（信号）

```typescript
import { signal } from '@type-dom/signals';

const count = signal(0);
count.get();  // 读取值（建立依赖）
count.set(1); // 设置值（触发更新）
```

**特性**: 
- ✅ 自动依赖追踪
- ✅ 精确更新
- ✅ 批量更新支持

### Computed（计算属性）

```typescript
import { signal, computed } from '@type-dom/signals';

const firstName = signal('John');
const lastName = signal('Doe');

const fullName = computed(() => {
  return `${firstName.get()} ${lastName.get()}`;
});

fullName.get(); // "John Doe"
```

**特性**:
- ✅ 缓存计算结果
- ✅ 惰性求值
- ✅ 可选 setter

### Effect（副作用）

```typescript
import { signal, effect } from '@type-dom/signals';

const count = signal(0);

const stop = effect(() => {
  console.log('Count:', count.get());
});

count.set(1); // 自动重新执行

stop(); // 停止监听
```

**特性**:
- ✅ 自动依赖追踪
- ✅ 可停止/重启
- ✅ 作用域管理

## 🚀 快速开始

### 安装

```bash
npm install @type-dom/signals
```

### 基本使用

```typescript
import { signal, computed, effect } from '@type-dom/signals';

// 1. 创建信号
const count = signal(0);

// 2. 创建计算属性
const doubled = computed(() => count.get() * 2);

// 3. 创建副作用
effect(() => {
  console.log('Count:', count.get(), 'Doubled:', doubled.get());
});

// 4. 更新信号
count.set(5); // 输出：Count: 5 Doubled: 10
```
